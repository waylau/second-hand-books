# second-hand-books
Second hand books.二手书集市。你的闲置，正是人家所需。

如果你有限制的书籍，可以在这里售卖。步骤很简单:

## 1. 打开一个 issue

点击[新建issue](https://github.com/waylau/second-hand-books/issues/new)。

* issue 标题格式：书名，成色，价格

比如：Spring 实战，9新，20

## 2. 填写书籍的相关信息

格式如下：

* 书名：
* 出版社：
* 出版年月：
* 原价：
* 现价：
* 成色：
* 链接：
* 备注：

参考：<https://github.com/waylau/second-hand-books/issues/1>

## 3. 如果已经出售，请关闭 issue
